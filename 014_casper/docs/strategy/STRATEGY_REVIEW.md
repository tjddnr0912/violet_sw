### FVG 매매의 3가지 필수 조건

영상에서 캐스퍼 전략의 핵심으로 강조한 3가지 조건:

| # | 조건 | 설명 |
|---|------|------|
| 1 | **시간이 정해져야 함** | 뉴욕장 9:30 ET 개장 시점 (유동성 집중 시간) |
| 2 | **기준이 있어야 함** | 9:30 15분봉의 고가/저가 = 유동성 기준선 (지지/저항선이 아님) |
| 3 | **기준선 돌파 + FVG 동시 형성** | 몸통 돌파와 FVG가 함께 나타나야 유효. 기준선 없이 형성된 FVG는 무의미 |

> 기준선은 "지지/저항선"이 아니라 **유동성(Liquidity) 기준선**입니다.

### 보완 규칙: 시간 초과 포지션 관리

영상에서는 90분 윈도우 내 청산을 전제하지만, 손절에도 익절에도 도달하지 못한 채 11:00 AM(ET)을 넘길 수 있습니다.

**규칙: 11:00 AM 이후 손절을 손익분기가(수수료 포함)로 이동**

손익분기가는 단순 진입가가 아니라, **왕복 수수료를 포함한 가격**입니다.
진입가에 매도하면 수수료만큼 손해이므로, 수수료를 회수할 수 있는 가격으로 설정합니다.

```
손익분기가 = 진입가 × (1 + 왕복 수수료율)

예시:
  진입가: $562.00
  왕복 수수료율: 0.18% (0.09% × 2)

  손익분기가 = $562.00 × 1.0018 = $563.01
```

```
9:50  진입 (TQQQ $43.00, 손절 $42.30, 익절 $44.40)  ← R:R 1:2
      손익분기가: $43.08

시나리오 A: 11:00 전에 익절/손절 도달 → 정상 청산
시나리오 B: 11:00 도달, 현재가 $43.20 (익절/손절 미도달)
  → 손절을 $43.08(손익분기가)로 이동
  → 이후 $43.08에 닿으면 수수료 포함 본전 청산, 익절에 닿으면 수익 청산
시나리오 C: 15:50(ET) 장 마감 10분 전까지도 미청산
  → 시장가 강제 청산 (오버나잇 리스크 차단)
```

| 시점 | 동작 |
|------|------|
| ~11:00 AM (ET) | 원래 손절/익절 유지 |
| 11:00 AM 이후 | 손절을 **손익분기가(수수료 포함)**로 이동, 익절은 유지 |
| 15:50 PM (ET) | 미청산 시 **시장가 강제 청산** (장 마감 10분 전) |

이 방식의 이점:
- 90분 이후에도 **수수료를 포함하여 손해를 보지 않음**
- 추세가 계속되면 익절까지 수익 실현 가능
- 오버나잇 포지션 보유 위험 완전 차단

### 정확한 전략 명칭

이 전략은 독자적인 신규 전략이 아니라 **세 가지 기존 방법론의 조합**입니다:

```
캐스퍼 매매법 = ORB (기준선) + ICT FVG (진입 트리거) + Pullback Entry (실행) + 고정 R:R (익절)
```

| 구성요소 | 원래 방법론 | 출처 |
|---------|-----------|------|
| 15분 Opening Range 기준선 | **Opening Range Breakout (ORB)** | Toby Crabel (1990) |
| Fair Value Gap 확인 | **ICT (Inner Circle Trader)** | Michael Huddleston (2016~) |
| FVG 되돌림 진입 | **Break-Retest-Continue** 패턴 | 고전적 Price Action |
| 고정 손익비 익절 | **Fixed R:R Money Management** | 일반적 자금 관리 |

## 2. 각 구성요소 유효성 평가

### 2.1 Opening Range Breakout (ORB)

**학술적 근거: 강함**

- Crabel (1990) "Day Trading with Short Term Price Patterns and Opening Range Breakout"에서 체계화
- 후속 연구: Linda Raschke & Larry Connors (Street Smarts, 1995)

**백테스트 데이터:**

| 출처 | 대상 | 승률 | Profit Factor |
|------|------|------|---------------|
| QuantifiedStrategies | S&P 500 (5분 ORB) | 65% | 2.0 |
| tosindicators | AAPL (15분 ORB) | 53~73% | - |
| Unger Academy | Nasdaq | 시간에 따라 감소 | 필터 필요 |

**15분 Opening Range의 특성:**
- 5분: 빠르지만 거짓 돌파 위험 높음
- **15분: 승률 면에서 가장 우수**, 데이트레이더 표준
- 30분: 가장 신뢰도 높지만 진입 기회 감소

**핵심 발견:** 개장 범위가 좁을수록 돌파 후 지속성이 강함 (NR4, NR7 패턴)

### 2.2 Fair Value Gap (FVG)

**학술적 근거: 취약 (실전적 관찰은 존재)**

- ICT(Michael Huddleston)가 YouTube 강의로 대중화
- 체계적인 학술 논문 없음
- FVG = 3개 연속 캔들에서 1번째 고가 ~ 3번째 저가 사이의 가격 공백

**실증 데이터 (Edgeful, YM 30분 차트):**
- Bullish FVG 69%가 동일 세션 내 미충족 유지
- Bearish FVG 68%가 동일 세션 내 미충족 유지
- 전체 FVG의 70~80%는 결국 채워짐
- 컨플루언스 결합 시 60~70%+ 승률 달성 가능

**핵심 판단:** FVG 자체보다 **ORB 레벨 + FVG의 위치 일치**가 실제 엣지. FVG가 ORB 기준선 근처에 형성될 때 가장 강한 신호.

### 2.3 Pullback Entry (되돌림 진입)

**학술적 근거: 강함**

- 돌파 직후 진입 대비 되돌림 후 진입이 통계적으로 유리
- R:R이 개선됨 (손절 거리 감소)
- 추격 매수 방지 구조

### 2.4 R:R (Risk-Reward Ratio, 손익비)

R:R은 **내가 감수하는 손실 대비 기대하는 수익의 비율**입니다.
이 전략에서 R:R은 **고정 금액이 아니라, 매 거래마다 손절폭에 비례하여 결정**됩니다.

#### 계산 방식

```
1. 손절가 결정: FVG 형성 캔들(제2캔들)의 이전 캔들(제1캔들) 저점
2. 진입가 결정: FVG 구간으로 되돌림이 왔을 때의 매수가
3. 1R = 진입가 - 손절가 (매 거래마다 다름)
4. 익절가 = 진입가 + (1R × 2 또는 3)
```

#### 구체적 예시

```
예시 A: 손절폭이 좁은 경우
  FVG 제1캔들 저점: $561.20 → 손절가
  진입가 (FVG 되돌림): $562.00
  1R = $562.00 - $561.20 = $0.80

  R:R 1:2 → 익절가 = $562.00 + ($0.80 × 2) = $563.60
  R:R 1:3 → 익절가 = $562.00 + ($0.80 × 3) = $564.40

예시 B: 손절폭이 넓은 경우
  FVG 제1캔들 저점: $559.50 → 손절가
  진입가 (FVG 되돌림): $562.00
  1R = $562.00 - $559.50 = $2.50

  R:R 1:2 → 익절가 = $562.00 + ($2.50 × 2) = $567.00
  R:R 1:3 → 익절가 = $562.00 + ($2.50 × 3) = $569.50
```

**핵심: 손절폭이 좁을수록 익절 목표도 가까워지고, 넓을수록 멀어집니다.**
손절폭이 너무 넓으면(예: ORB 범위의 50% 초과) 90분 내 익절 도달이 어려우므로 해당 세팅은 스킵합니다.

#### FVG 캔들 구조

```
제1캔들    제2캔들(displacement)    제3캔들
  │            │                      │
  │    ┌───────┤                      │
  │    │  강한  │          ┌───────┐   │
  ├──┐ │  상승  │          │       │   │
  │  │ │  캔들  │          │       │   │
  └──┘ │       │          └───────┘   │
       └───────┘                      │
  ↑                    ↑
  손절가              FVG 영역
  (제1캔들 저점)      (제1캔들 고가 ~ 제3캔들 저가)
```

#### R:R이 높으면 승률이 낮아도 수익 가능

```
R:R 1:3, 10번 거래, 승률 40% (4승 6패)
  수익: 4 × 3R = 12R
  손실: 6 × 1R = 6R
  순이익: +6R ✓  (수수료 제외)
```

**이 전략의 기본 R:R: 1:2** (백테스트 최적값, 상세 근거는 4.3절 및 7절 백테스트 결과 참조)

| R:R | 수수료 없을 때 손익분기 승률 | 한투 수수료 포함 손익분기 승률 |
|-----|-------------------------|---------------------------|
| **1:2** | **33%** | **~40%** ← 채택 (TQQQ 백테스트 승률 43.5%에서 수익 확인) |
| 1:3 | 25% | ~35% |

> R:R 1:3 → 1:2 변경 사유:
> SPY R:R 1:3 백테스트에서 77%가 본전 청산. R:R 1:2로 변경 시에도 SPY는 68% 본전 청산.
> TQQQ R:R 1:2 백테스트에서 본전 청산 43%, 승률 43.5%, PF 5.46으로 확인.
> 3x 레버리지로 움직임이 증폭되어 R:R 1:2가 90분 윈도우 내 도달 가능한 현실적 목표.

## 3. 전략의 장점과 한계

### 장점

1. **명확한 규칙 기반** - 진입/손절/익절이 사전 정의, 감정 개입 최소화
2. **높은 유동성 구간** - 뉴욕 개장 초반은 하루 중 거래량 최대 집중 구간
3. **되돌림 대기** - 불리한 진입 회피 구조
4. **짧은 트레이딩 윈도우** - 90분만 모니터링

### 한계

1. **FVG 미형성** - 모든 돌파에서 FVG가 형성되지 않음 → 세팅 부재일 다수
2. **페이크아웃 취약** - ORB 레벨에서의 거짓 돌파에 취약
3. **손절 폭 변동** - FVG 캔들 이전 저점이 때로 너무 넓어 R:R 1:3 미충족
4. **조기 익절** - 90분 제한으로 추세 지속 시 수익 제한 (단, 시간 초과 시 손익분기 이동 규칙으로 보완)

### 시장 조건별 작동 여부

| 조건 | 잘 작동 | 실패 |
|------|---------|------|
| 시장 성격 | 뚜렷한 추세, 강한 개장 모멘텀 | 양방향 급등락, 갭 혼재 |
| VIX | 15~25 (정상 변동성) | 30+ (패닉), 12 이하 (무방향) |
| 종목 | 높은 유동성, 명확한 베타 | 얇은 호가, 섹터 이슈 |
| 뉴스 | 뉴스 없는 날 순수 가격 액션 | FOMC/CPI/실적 발표 당일 |
| FVG 위치 | ORB 레벨과 밀접하게 겹침 | ORB 레벨과 멀리 떨어진 FVG |

## 4. 누락된 요소 (Critical Gaps)

### 4.1 종목 선택 기준 (완전히 부재)

**최소 필터 요건:**

| 기준 | 권장값 | 설명 |
|------|--------|------|
| 일평균 거래량 | 500만주+ | 원활한 체결 보장 |
| 일평균 달러 거래량 | $1억+ | 유동성 확보 |
| ATR | $0.50+ (일봉) | 충분한 변동성 |
| Pre-Market 거래량 | 10만주+ (9AM 기준) | 당일 관심도 확인 |
| Relative Volume | 1.5x+ | 10일 평균 대비 |
| Gap % | 대형주 3%+, 소형주 10%+ | 모멘텀 확인 |
| 카탈리스트 | 필수 | 뉴스, 실적 등 |

**채택:** TQQQ/SQQQ → 추세 필터(QQQ 일봉)와 기초지수 일치, 3x 레버리지로 FVG 목표 도달률 극대화
> TQQQ ~$43/주, SQQQ ~$28/주. $500 자본으로 10주+ 매수 가능.
> 백테스트 결과 SPY 대비 압도적 성과 확인 (7절 참조).

### 4.2 시장 방향성(Market Bias) 필터 (가장 치명적 누락)

ORB 상단 돌파 + 매수 세팅이지만, 시장 전체가 하락 압력 하에 있으면 성공 확률 절반 이하.

**3계층 필터 구조:**

**모든 시간은 미국 동부시간(ET) 기준입니다. 한국시간(KST) = ET + 14시간 (서머타임) / ET + 13시간 (겨울).**

#### Layer 1: 추세 필터 — 매매 방향 결정

**확인 시점:** 미국장 마감 후 (ET 16:00 이후 / KST 다음날 06:00 이후)

| 필터 | 확인 방법 | 적용 규칙 |
|------|----------|----------|
| QQQ 일봉 20일 MA | 종가 기준 위/아래 | 위 = TQQQ Long, 아래 = SQQQ Long |
| QQQ 일봉 50일 MA | 종가 기준 위/아래 | 20MA + 50MA 모두 아래 → SQQQ 전환 |
| 주간 추세 | 직전 5일간 고점/저점 방향 | Higher High/Higher Low = 상승, 반대 = 하락 |

**공매도/선물은 사용하지 않습니다.** 하락장에서는 SQQQ(3x 인버스)를 매수(Long)하여 대응합니다.

**매매 대상 ETF:**

추세 필터는 **QQQ 일봉** 기준으로 판단하고, 거래는 3x 레버리지 ETF로 실행합니다.
TQQQ/SQQQ는 QQQ(Nasdaq-100)를 3배로 추종하므로, 추세 판단 기준과 거래 대상의 기초지수가 일치합니다.

| 시장 방향 | 매매 대상 | 비고 |
|----------|----------|------|
| 상승 추세 (QQQ 20MA 위) | **TQQQ** Long (~$43/주) | Nasdaq-100 3x Bull |
| 하락 추세 (QQQ 20MA 아래) | **SQQQ** Long (~$28/주) | Nasdaq-100 3x Bear |
| 20MA 부근 (±0.5%) | TQQQ 또는 SQQQ, 수량 50% 축소 | 방향 불확실 구간 |

**TQQQ/SQQQ 쌍의 이점:**
- TQQQ 일평균 거래량 ~1.2억주, SQQQ ~1.1억주 → 양방향 모두 초고유동성
- 두 ETF 모두 ProShares 운용, 동일 기초지수(Nasdaq-100) 추종
- 당일 90분 내 청산하므로 3x 레버리지의 일별 리밸런싱 괴리 무관
- $43/$28 단가로 소형 계좌에서도 충분한 수량 매수 가능

#### Layer 2: 당일 컨텍스트 — Go/No-Go 결정

**확인 시점:** Pre-market (ET 8:00~9:25 / KST 22:00~23:25)

| 필터 | 확인 방법 | 적용 규칙 |
|------|----------|----------|
| VIX 수준 | 전일 종가 | 12~30: 정상 진행 / 30+: 사이즈 50% 또는 스킵 / 12 이하: 스킵 |
| 매크로 이벤트 | 경제 캘린더 | FOMC 결정일, CPI, NFP 당일 → **무조건 스킵** |
| Pre-market 갭 | QQQ 선물(NQ) vs 전일 종가 | 갭 방향이 Layer 1 추세와 일치 시 확신 ↑ / 역방향 갭 → 주의 |
| Pre-market 볼륨 | ET 8:00~9:30 NQ 선물 거래량 | 평균 대비 2x+ → 강한 모멘텀 기대 / 평균 이하 → 약한 움직임 |

**No-Trade 일:**
- FOMC 금리 결정일 (발표 전후 모두)
- CPI/PPI 발표일 (ET 8:30 AM 발표 → 개장 직후 변동성 과대)
- 분기 옵션 만기일 (Triple/Quadruple Witching)
- VIX 30 초과

#### Layer 3: 실시간 확인 — 진입 품질 판단

**확인 시점:** ORB 형성 중 (ET 9:30~9:45 / KST 23:30~23:45)

| 필터 | 확인 방법 | 적용 규칙 |
|------|----------|----------|
| ORB 범위 크기 | 고가-저가 vs 20일 평균 ATR | ORB > ATR × 1.5 → 이미 과도한 움직임, 스킵 |
| 갭 방향 정합성 | 갭 방향 vs 돌파 방향 | 갭업 + 상단 돌파 = 정합 ✓ / 갭업 + 하단 돌파 = 부정합 (낮은 확률) |
| 시장 너비 | TICK 지표 (NYSE Tick) | TICK > +500 = 매수 우세 / TICK < -500 = 매도 우세 |

#### 타임라인 요약 (KST 기준, 서머타임)

```
KST 06:00        미국장 마감 → Layer 1 추세 확인
KST 22:00~23:25  Pre-market  → Layer 2 Go/No-Go 판단
KST 23:30~23:45  장 개장     → ORB 형성 (15분), Layer 3 확인
KST 23:45~01:00  매매 구간   → 세팅 발생 시 1회 진입, 결과 후 종료
```

**최종 판단 흐름:**
```
Layer 1 (추세 방향) → Layer 2 (당일 Go/No-Go) → Layer 3 (진입 품질)
         ↓                      ↓                        ↓
   일반ETF/인버스ETF       Trade/Skip              Enter/Wait/Skip
```

#### 소수점 매매 제약 (중요)

**KIS Open API에서 소수점 매매는 불가능합니다. 최소 주문 단위는 1주(정수)입니다.**

- 소수점 투자는 미니스탁(앱 전용) 서비스로만 가능
- API의 `ord_qty` 파라미터는 정수만 허용
- 008_stock_trade_us 프로젝트에서도 `qty: int` 정수 검증 확인됨

TQQQ ~$43/주, SQQQ ~$28/주이므로 소형 계좌에서도 충분한 수량 매수 가능.

**→ 초기 자본 $200으로도 TQQQ 4주, SQQQ 7주 매수 가능. 권장 초기 자본: $500+.**

#### 수수료 구조

**전제: 달러를 미리 환전하여 보유. 매매 시 환전이 발생하지 않으므로 환전 스프레드 없음.**

| 항목 | 비율 | TQQQ $43 × 30주 기준 |
|------|------|---------------------|
| 매수 수수료 | 0.09% (이벤트) | $1.16 |
| 매도 수수료 | 0.09% (이벤트) | $1.16 |
| 환전 스프레드 | 없음 (달러 보유) | $0.00 |
| SEC Fee | ~0.002% (매도만) | ~$0.03 |
| **왕복 합계** | **0.18%** | **~$2.35** |

> 이벤트 수수료: 뱅키스 신규/휴면 고객 최초 3개월 무료 → 이후 9개월 0.09% → 거래 실적 시 자동 연장.
> 기본 수수료(0.25%)로 복귀 시 왕복 0.50%로 증가 → 이벤트 유지 필수.

#### 수수료가 손익에 미치는 영향

R:R은 비율 기반이므로, 수수료 영향도 비율로 분석합니다:

```
왕복 수수료율: 0.18%
손익분기가 = 진입가 × (1 + 0.0018)

TQQQ $43 기준:
  손익분기가 = $43 × 1.0018 = $43.08 (진입가 대비 +$0.08)
```

**손익분기 승률 (비율 기반이므로 종목/계좌 크기 무관):**

| R:R | 수수료 없을 때 | 한투 0.09% (달러 보유) |
|-----|-------------|---------------------|
| **1:2** | **33%** | **~40%** ← 채택 |
| 1:3 | 25% | ~35% |
| 1:4 | 20% | ~27% |

R:R 1:2 + 한투 이벤트 수수료 기준, **승률 40% 이상이면 수익**. TQQQ 백테스트 승률 43.5%에서 확인.

### 4.4 포지션 사이징

#### 운용 환경

| 항목 | 내용 |
|------|------|
| 초기 계좌 | $500+ 권장 (TQQQ 10주+ 매수 가능) |
| 브로커 | 한국투자증권 KIS (REST API) |

| 매매 종목 | TQQQ (~$43/주, 상승) / SQQQ (~$28/주, 하락) |
| PDT 규칙 | **미적용** (한국 증권사는 미국 FINRA/PDT 규정 대상 아님) |
| 매매 횟수 제한 | 없음 (하루 1회는 전략 규칙이지 브로커 제한 아님) |

> 한국 증권사를 통한 미국주식 거래는 PDT 규칙(5일 내 4회+ 데이트레이드 시 $25,000 최소 자본)이
> 적용되지 않습니다. 이는 소형 계좌로 데이트레이딩을 할 때 큰 이점입니다.

#### 매수 수량 결정

API가 정수 주문만 지원하므로, 잔액으로 매수 가능한 **최대 정수 주수**를 매수합니다.

```
매수 수량 = floor(잔액 / 현재가)  ← 정수 내림

$500 계좌, TQQQ $43 기준:
  매수 가능: $500 / $43 = 11주 (나머지 $27 현금 대기)

$1,500 계좌, TQQQ $43 기준:
  매수 가능: $1,500 / $43 = 34주 (나머지 $38 현금 대기)

SQQQ $28 기준 (하락 추세):
  $1,500 / $28 = 53주
```

#### 리스크 (R:R 비율 기반으로 매번 다름)

```
TQQQ $43, 30주 보유, 손절폭 $0.70 (1R) 기준:
  손절 시 손실: 30 × $0.70 = $21.00
  익절 시 수익(1:2): 30 × $1.40 = $42.00
  왕복 수수료: $43 × 30 × 0.18% = $2.32

  순익절: $42.00 - $2.32 = +$39.68
  순손절: $21.00 + $2.32 = -$23.32

TQQQ $43, 30주 보유, 손절폭 $0.40 (1R) 기준:
  손절 시 손실: 30 × $0.40 = $12.00
  익절 시 수익(1:2): 30 × $0.80 = $24.00
  왕복 수수료: $43 × 30 × 0.18% = $2.32

  순익절: $24.00 - $2.32 = +$21.68
  순손절: $12.00 + $2.32 = -$14.32

→ TQQQ는 단가가 낮아 수수료 절대액($2.32)이 SPY($1.01) 대비 크지만,
  30주 보유로 손익 절대액도 커서 수수료 비율 영향은 동일(0.18%).
  최소 1R > $0.10 필요.
```

#### 계좌 성장에 따른 포지션 변화

```
$500   → TQQQ 11주   손절폭 $0.70 기준 월 기대수익(승률43%) +$15~20
$1,500 → TQQQ 34주   월 기대수익 +$50~60
$3,000 → TQQQ 69주   월 기대수익 +$100~120
$6,000 → TQQQ 139주  월 기대수익 +$200~240
```

> 수익률(%)은 계좌 크기와 무관하게 동일합니다. 수수료도 비율 기반(0.09%)이므로
> 계좌가 커져도 수익률(%)은 변하지 않고, 절대 금액만 증가합니다.

#### 매수 흐름 예시

```
Day 3: 잔액 $1,508.40, TQQQ $43.50 → 34주 매수
       ...
```

### 4.4 매매 횟수 및 연속 손절 대응

#### 원칙: 하루 1회 매매

영상 기준, 이 전략은 **하루 1번 세팅 → 1번 매매 → 끝**입니다.
90분 윈도우에서 세팅이 발생하면 진입하고, 손절이든 익절이든 결과가 나오면 당일 종료.
세팅이 안 나오면 그날은 무거래.

```
9:45  세팅 발생 → 진입
10:15 손절 or 익절 → 당일 매매 종료. 추가 진입 없음.
```

**하루 1회의 이점:**
- 감정적 복수 매매(Revenge Trading) 원천 차단
- PDT 규칙 부담 감소 (주 5회 이하로 유지 가능)
- 판단 피로 최소화

#### Circuit Breaker: 주 단위 초기화

모든 제약은 **매주 월요일(첫 거래일)에 초기화**됩니다.

| 규칙 | 적용 | 초기화 |
|------|------|--------|
| 주간 연속 3회 손절 | 해당 주 잔여 거래일 매매 중단 | 다음 주 월요일 리셋 |
| 주간 손실 누적 > 계좌 3% | 해당 주 잔여 거래일 매매 중단 | 다음 주 월요일 리셋 |

```
예시: 월~수 3일 연속 손절
  → 목, 금 매매 중단 (관망만)
  → 다음 주 월요일부터 정상 거래 재개

예시: 월 손절, 화 익절, 수 손절, 목 손절
  → 연속 아님 (화에 끊김) → 제약 없음, 금요일 정상 거래
```

주말은 자연스러운 리셋 구간입니다. 주중에 제약이 걸려도 다음 주에는 새로운 시장 조건으로 시작하므로 이전 주의 손절 이력을 이월하지 않습니다.

중단 기간에는 차트를 관찰만 하고, 세팅이 작동했을지를 기록하여 시장 적합성을 재확인합니다.

**뉴욕장 첫 90분 특성:**

| 시간대 (ET) | 특성 |

### 7.4 슬리피지(Slippage)에 대한 설명

**슬리피지 = 주문한 가격과 실제 체결된 가격의 차이**

본 백테스트는 슬리피지를 반영하지 않았습니다. 실전에서는 다음과 같은 추가 비용이 발생합니다:

| 요인 | 설명 | TQQQ/SQQQ 예상 영향 |
|------|------|---------------------|
| 호가 스프레드 (Bid-Ask Spread) | 매수는 Ask, 매도는 Bid에 체결 | ~$0.01/주 (고유동성으로 최소) |
| 시장 충격 (Market Impact) | 주문 체결 시 가격 밀림 | 30주 수준에서 무시 가능 |
| 지연 (Latency) | 주문~체결 사이 가격 변동 | ~$0.01-0.03/주 (변동성 구간) |
| **왕복 추정** | | **~$0.02-0.04/주** |

```
TQQQ/SQQQ 평균 25주 기준 슬리피지 추정:
  왕복: 25주 × $0.03 = $0.75/거래
  23거래: $0.75 × 23 = $17.25

  백테스트 순손익 $168.20 - 슬리피지 $17.25 = $150.95 (여전히 양수)
  수익률: +11.21% → +10.06% (영향 제한적)
```

TQQQ/SQQQ 모두 일평균 거래량 1억주+로 레버리지 ETF 중 최고 유동성이므로 슬리피지 영향 최소.
